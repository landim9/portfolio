## 📋 Sobre o Projeto

Portfólio pessoal construído do zero com HTML, CSS e JavaScript sem frameworks ou bibliotecas externas. O objetivo foi criar uma apresentação profissional que demonstra os projetos, habilidades e formas de contato de Rafael David, com foco em performance e código limpo.

---

## ✨ Seções

- **Home** — Apresentação com nome, cargo e links principais
- **Sobre** — Background profissional e acadêmico
- **Projetos** — Cards com descrição, stack e links para GitHub e deploy
- **Habilidades** — Stack técnica organizada por categoria
- **Contato** — Links para LinkedIn, GitHub e e-mail

## 🚀 Rodando Localmente

### Opção 1 — Abrir diretamente

Abra o arquivo `index.html` no seu navegador.

### Opção 2 — Servidor local (recomendado)

Com a extensão **Live Server** no VS Code:

1. Abra a pasta no VS Code
2. Clique com o botão direito em `index.html`
3. Selecione **Open with Live Server**

Ou via terminal com `npx`:

```bash
npx serve .
```

Acesse [http://localhost:3000](http://localhost:3000)
